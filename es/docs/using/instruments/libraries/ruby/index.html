Ruby API Reference

The Ruby AAM library provides a native Ruby extension for high-performance configuration parsing.

Main Namespace​

NamespaceDescription
AamRb::AAMThe primary object for interacting with an AAM document.

Class Methods​

Static MethodReturnDescription
AamRb::AAM.newAamRb::AAMCreates a new empty document.
AamRb::AAM.parse(content)AamRb::AAMCreates a document from a string.
AamRb::AAM.load(path)AamRb::AAMCreates a document from a file path.

Instance Methods​

MethodReturnDescription
get(key)`StringNil`
keysArray<String>Returns all keys in the document.
to_mapHash<String, String>Transforms library data into a Ruby hash.
find(query)Hash<String, String>Advanced forward/reverse lookup by key or value.
deep_search(pattern)Hash<String, String>Recursive reference lookup with pattern support.
reverse_search(value)Array<String>Search for all keys matching a given value.
schema_namesArray<String>Returns all registered schema names.
type_namesArray<String>Returns all registered type names.

Error Model​

  • Parse and load failures raise native RuntimeError exceptions from the core implementation.
nota

Existing README documentation for the AAM Ruby library provides lightweight helper styles. The core objective of the native extension API remains focused on AamRb::AAM.
