C# / .NET API Reference

The C# library provides a native .NET interface for interacting with AAM files, wrapping the core Rust implementation through P/Invoke.

Main Type​

TypeDescription
AamDocumentImplements IDisposable and provides the primary API for interaction.

Construction​

MethodDescription
new AamDocument()Creates an empty AAM document.
AamDocument.Parse(string content)Static factory method creating an AamDocument from a source string.
AamDocument.Load(string path)Static factory method creating an AamDocument from a file path.

Core Methods​

MethodReturnDescription
Format(string content)stringFormats the input AAM content according to standard rules.
Merge(string content)voidMerges new content into the current document instance.
RecoverSimple(string content)voidSafely recovers content from corrupted sources.
FindObj(string key)string?Forward/Reverse lookup for a given key or value.
FindKey(string value)string?Explicit reverse lookup for a value.
FindDeep(string key)string?Deep reference resolution for a key.

Lifecycle​

Property/MethodDescription
bool IsClosedIndicates whether the native handle has been disposed.
void Dispose()Releases all native resources associated with the document.
astuce

Always use using statements with AamDocument to ensure deterministic cleanup of native resources.

Error Model​

The C# library uses standard .NET exception handling:

  • Throws AamException for native errors during parsing, loading, formatting, etc.
  • Throws ObjectDisposedException if you attempt to use a document after calling Dispose().
