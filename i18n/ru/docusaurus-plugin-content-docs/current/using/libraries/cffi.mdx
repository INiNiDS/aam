# Справочник API C FFI

Источник истины: `include/aam.h`. Этот API предоставляет стабильный интерфейс C для интеграции AAM в другие языки или системы.

## Жизненный цикл дескриптора

| Функция | Описание |
|:---|:---|
| `AamlHandle *aam_new(void)` | Создает новый дескриптор AAML. |
| `void aam_free(AamlHandle *handle)` | Освобождает ранее созданный дескриптор. |

## Разбор и загрузка

| Функция | Описание |
|:---|:---|
| `int aam_parse(AamlHandle *handle, const char *content)` | Разбирает содержимое AAM из строки. |
| `int aam_load(AamlHandle *handle, const char *path)` | Загружает и разбирает файл AAM с диска. |
| `int aam_merge(AamlHandle *handle, const char *content)` | Объединяет новое содержимое с существующим дескриптором. |
| `int aam_recover_simple(AamlHandle *handle, const char *content)` | Безопасный разбор для восстановления поврежденных входных данных. |

**Конвенция возврата:** `0` при успехе, ненулевое значение при ошибке.

## Поиск (Lookup)

| Функция | Описание |
|:---|:---|
| `char *aam_find_obj(const AamlHandle *handle, const char *key)` | Прямой поиск по ключу или обратный поиск по значению. |
| `char *aam_find_key(const AamlHandle *handle, const char *value)` | Явный обратный поиск по значению. |
| `char *aam_find_deep(const AamlHandle *handle, const char *key)` | Рекурсивное разрешение ссылок. |

:::important
Возвращаемые строки принадлежат Rust и должны быть освобождены через `aam_string_free` во избежание утечек памяти.
:::

## Память и ошибки

| Функция | Описание |
|:---|:---|
| `void aam_string_free(char *s)` | Освобождает строку, возвращенную любой из функций поиска. |
| `const char *aam_last_error(const AamlHandle *handle)` | Возвращает сообщение о последней ошибке в виде заимствованной строки. |

:::warning
`aam_last_error` возвращает внутренний заимствованный указатель. **Не освобождайте его.**
:::
