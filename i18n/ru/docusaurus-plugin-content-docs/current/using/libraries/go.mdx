# Справочник API Go

Библиотека AAM для Go является тонкой оберткой над основной библиотекой на Rust, предоставляя нативный интерфейс Go для эффективного разбора конфигураций.

## Основной тип

| Тип | Описание |
|:---|:---|
| `AAM` | Основная структура, используемая для взаимодействия с документами AAM. |

---

## Конструкторы

| Метод | Возвращает | Описание |
|:---|:---|:---|
| `New()` | `(*AAM, error)` | Создает новый пустой документ. |
| `Parse(content string)` | `(*AAM, error)` | Создает документ из строки исходного кода. |
| `Load(path string)` | `(*AAM, error)` | Создает документ из пути к файлу. |

---

## Методы экземпляра

| Метод | Возвращает | Описание |
|:---|:---|:---|
| `Format(content string)` | `(string, error)` | Форматирует содержимое исходного кода AAM. |
| `Get(key string)` | `(string, bool)` | Прямой поиск по ключу. |
| `Find(query string)` | `map[string]string` | Продвинутый прямой/обратный поиск по ключу или значению. |
| `DeepSearch(pattern string)` | `map[string]string` | Рекурсивный поиск ссылок с поддержкой шаблонов. |
| `ReverseSearch(value string)` | `[]string` | Поиск всех ключей, соответствующих заданному значению. |
| `SchemaNames()` | `[]string` | Возвращает список всех зарегистрированных имен схем. |
| `TypeNames()` | `[]string` | Возвращает список всех зарегистрированных имен типов. |
| `LastError()` | `string` | Извлекает сообщение о последней ошибке из нативного дескриптора. |
| `Close()` | `void` | Закрывает нативный дескриптор и освобождает память. |

---

:::note
- Метод `Close()` идемпотентен и должен вызываться после завершения работы с документом.
- Методы на закрытом дескрипторе возвращают пустые значения или ошибки в зависимости от конкретного метода.
:::
