# Справочник API PHP

Библиотека AAM для PHP предоставляет удобный интерфейс для взаимодействия с файлами конфигурации AAM через PHP FFI.

## Основной класс

| Класс | Описание |
|:---|:---|
| `AamPhp` | Основной объект для взаимодействия с документом AAM. |

---

## Создание

| Конструктор | Описание |
|:---|:---|
| `new AamPhp(?string $libraryPath = null)` | Создает новый экземпляр. Путь к библиотеке опционален и может быть использован для явного указания расположения нативной общей библиотеки. |

---

## Предоставляемый метод

| Метод | Возвращает | Описание |
|:---|:---|:---|
| `parseFindObj(string $content, string $query)` | `?string` | Разбирает входное содержимое и выполняет поиск в стиле `find_obj` по строке запроса. |

---

## Заметки времени выполнения

- Использует PHP FFI и требует наличия в системе установленной нативной общей библиотеки `aam_rs`.
- Путь к библиотеке может быть указан напрямую или будет разрешен из окружения/стандартных путей.
- Поддерживает откат к обратному поиску, если предоставленный запрос не является ключом.
