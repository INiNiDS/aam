# Справочник API Ruby

Библиотека AAM для Ruby предоставляет нативное расширение Ruby для высокопроизводительного разбора конфигураций.

## Основное пространство имен

| Пространство имен | Описание |
|:---|:---|
| `AamRb::AAM` | Основной объект для взаимодействия с документом AAM. |

---

## Методы класса

| Статический метод | Возвращает | Описание |
|:---|:---|:---|
| `AamRb::AAM.new` | `AamRb::AAM` | Создает новый пустой документ. |
| `AamRb::AAM.parse(content)` | `AamRb::AAM` | Создает документ из строки. |
| `AamRb::AAM.load(path)` | `AamRb::AAM` | Создает документ из пути к файлу. |

---

## Методы экземпляра

| Метод | Возвращает | Описание |
|:---|:---|:---|
| `get(key)` | `String | Nil` | Извлекает значение для заданного ключа. |
| `keys` | `Array<String>` | Возвращает все ключи в документе. |
| `to_map` | `Hash<String, String>` | Преобразует данные библиотеки в хеш Ruby. |
| `find(query)` | `Hash<String, String>` | Продвинутый прямой/обратный поиск по ключу или значению. |
| `deep_search(pattern)` | `Hash<String, String>` | Рекурсивный поиск ссылок с поддержкой шаблонов. |
| `reverse_search(value)` | `Array<String>` | Поиск всех ключей, соответствующих заданному значению. |
| `schema_names` | `Array<String>` | Возвращает все зарегистрированные имен схем. |
| `type_names` | `Array<String>` | Возвращает все зарегистрированные имен типов. |

---

## Модель ошибок

- Ошибки разбора и загрузки вызывают нативные исключения `RuntimeError` из основной реализации.

---

:::note
Существующая документация README для библиотеки AAM на Ruby предлагает облегченные стили вспомогательных методов. Основной целью API нативного расширения остается интерфейс `AamRb::AAM`.
:::
