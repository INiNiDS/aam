# Справочник API C# / .NET

Библиотека C# предоставляет нативный интерфейс .NET для взаимодействия с файлами AAM, оборачивая основную реализацию на Rust через P/Invoke.

## Основной тип

| Тип | Описание |
|:---|:---|
| `AamDocument` | Реализует `IDisposable` и предоставляет основной API для взаимодействия. |

---

## Создание

| Метод | Описание |
|:---|:---|
| `new AamDocument()` | Создает пустой документ AAM. |
| `AamDocument.Parse(string content)` | Статический фабричный метод, создающий `AamDocument` из строки исходного кода. |
| `AamDocument.Load(string path)` | Статический фабричный метод, создающий `AamDocument` из пути к файлу. |

---

## Основные методы

| Метод | Возвращает | Описание |
|:---|:---|:---|
| `Format(string content)` | `string` | Форматирует входное содержимое AAM в соответствии со стандартными правилами. |
| `Merge(string content)` | `void` | Объединяет новое содержимое с текущим экземпляром документа. |
| `RecoverSimple(string content)` | `void` | Безопасно восстанавливает содержимое из поврежденных источников. |
| `FindObj(string key)` | `string?` | Прямой/обратный поиск для заданного ключа или значения. |
| `FindKey(string value)` | `string?` | Явный обратный поиск для значения. |
| `FindDeep(string key)` | `string?` | Глубокое разрешение ссылок для ключа. |

---

## Жизненный цикл

| Свойство/Метод | Описание |
|:---|:---|
| `bool IsClosed` | Указывает, был ли закрыт нативный дескриптор. |
| `void Dispose()` | Освобождает все нативные ресурсы, связанные с документом. |

:::tip
Всегда используйте блоки `using` с `AamDocument`, чтобы гарантировать детерминированную очистку нативных ресурсов.
:::

---

## Модель ошибок

Библиотека C# использует стандартную обработку исключений .NET:

- Выбрасывает `AamException` при возникновении нативных ошибок во время разбора, загрузки, форматирования и т. д.
- Выбрасывает `ObjectDisposedException`, если вы пытаетесь использовать документ после вызова `Dispose()`.
