# Справочник API Node.js

Библиотека AAM для Node.js предоставляет высокоуровневый JS-интерфейс для продвинутого разбора конфигураций через нативные привязки.

## Функции верхнего уровня

| Функция | Возвращает | Описание |
|:---|:---|:---|
| `parse(content: string)` | `AAM` | Создает документ из строки исходного кода. |
| `load(path: string)` | `AAM` | Создает документ из пути к файлу на диске. |
| `recoverSimple(content: string)` | `AAM` | Безопасно восстанавливает содержимое из поврежденных источников. |
| `version()` | `string` | Возвращает текущую версию библиотеки AAM. |

---

## Основной класс

| Класс | Описание |
|:---|:---|
| `AAM` | Основной объект для взаимодействия с документом AAM. |

---

## Методы экземпляра

| Метод | Возвращает | Описание |
|:---|:---|:---|
| `merge(content: string)` | `void` | Объединяет новое содержимое с текущим экземпляром документа. |
| `mergeContent(content: string)` | `void` | Алиас для `merge`. |
| `mergeFile(path: string)` | `void` | Объединяет содержимое файла с текущим экземпляром документа. |
| `findObj(key: string)` | `string \| null` | Прямой/обратный поиск по заданному ключу или значению. |
| `findKey(value: string)` | `string \| null` | Явный обратный поиск для значения. |
| `findDeep(key: string)` | `string \| null` | Глубокое разрешение ссылок для ключа. |
| `findList(key: string)` | `string[] \| null` | Возвращает список значений, если ключ сопоставлен с ним. |
| `findObject(key: string)` | `Record<string, string> \| null` | Возвращает объект, если ключ сопоставлен с ним. |
| `keys()` | `string[]` | Возвращает все ключи в документе. |
| `toMap()` | `Record<string, string>` | Преобразует все данные документа в JS-объект. |
| `validateValue(typeName: string, value: string)` | `void` | Базовая валидация значения в соответствии с зарегистрированными типами. |
| `close()` | `void` | Закрывает нативный дескриптор и освобождает нативные ресурсы. |
| `isClosed()` | `boolean` | Указывает, был ли закрыт документ. |

:::tip
Для удобства `AAML`, `JsAam` и `JsAaml` используются как алиасы для `AAM`.
