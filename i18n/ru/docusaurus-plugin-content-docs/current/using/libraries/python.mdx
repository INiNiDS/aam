# Справочник API Python

Библиотека AAM для Python предоставляет нативный интерфейс Python для продвинутого разбора конфигураций через привязки на базе PyO3.

## Основной класс

| Класс | Описание |
|:---|:---|
| `AAM` | Основной объект для взаимодействия с документом AAM. |

---

## Создание

| Статический метод / Конструктор | Возвращает | Описание |
|:---|:---|:---|
| `AAM()` | `AAM` | Создает новый пустой документ. |
| `AAM.parse(content: str)` | `AAM` | Создает документ из строки исходного кода. |
| `AAM.load(path: str)` | `AAM` | Создает документ из пути к файлу на диске. |

---

## Статический вспомогательный метод

| Статический метод | Возвращает | Описание |
|:---|:---|:---|
| `AAM.lsp_assist(content: str)` | `tuple[list[str], str | None]` | Возвращает диагностическую информацию и отформатированный код для поддержки IDE. |

---

## Методы экземпляра

| Метод | Возвращает | Описание |
|:---|:---|:---|
| `format(content: str)` | `str` | Форматирует содержимое исходного кода AAM. |
| `format_range(content: str, start_line: int, end_line: int)` | `str` | Форматирует определенный диапазон строк в коде. |
| `get(key: str)` | `str | None` | Прямой поиск по ключу. |
| `keys()` | `list[str]` | Возвращает все ключи в документе. |
| `to_dict()` | `dict[str, str]` | Преобразует данные библиотеки в словарь Python. |
| `find(query: str)` | `dict[str, str]` | Продвинутый прямой/обратный поиск по ключу или значению. |
| `deep_search(pattern: str)` | `dict[str, str]` | Рекурсивный поиск ссылок с поддержкой шаблонов. |
| `reverse_search(target_value: str)` | `list[str]` | Поиск всех ключей, соответствующих заданному значению. |
| `schema_names()` | `list[str]` | Возвращает список всех зарегистрированных имен схем. |
| `type_names()` | `list[str]` | Возвращает список всех зарегистрированных имен типов. |
| `close()` | `None` | Вручную освобождает все нативные ресурсы. |
| `is_closed()` | `bool` | Указывает, был ли закрыт документ. |

---

## Методы протокола Python

- `__repr__`: Человекочитаемое представление объекта.
- `__len__`: Количество ключей в документе.
- `__contains__`: Прямая проверка наличия ключа с помощью `key in aam`.
- `__getitem__`: Доступ в стиле словаря с помощью `aam[key]`.
