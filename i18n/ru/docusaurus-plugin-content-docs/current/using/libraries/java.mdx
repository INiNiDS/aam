# Справочник API Java / Kotlin

JVM-библиотека AAM предоставляет нативный разбор конфигураций для JVM через JNI, обернутый в удобный для разработчиков интерфейс на Kotlin, полностью совместимый с Java.

## Основной тип

| Тип | Описание |
|:---|:---|
| `com.rustgames.aam.AamDocument` | Реализует `AutoCloseable` и предоставляет основной API для взаимодействия с документом. |

---

## Создание

| Статический метод | Возвращает | Описание |
|:---|:---|:---|
| `AamDocument.parse(String content)` | `AamDocument` | Создает документ из строки. |
| `AamDocument.load(String path)` | `AamDocument` | Создает документ из пути к файлу. |

---

## Методы экземпляра

| Метод | Возвращает | Описание |
|:---|:---|:---|
| `reload(String content)` | `void` | Перезаписывает текущее содержимое документа новым исходным кодом. |
| `get(String key)` | `String?` | Извлекает значение для заданного ключа. |
| `deepSearch(String pattern)` | `Map<String, String>` | Рекурсивное разрешение ссылок. |
| `reverseSearch(String value)` | `List<String>` | Находит все ключи, которые сопоставлены с конкретным значением. |
| `schemaNames()` | `List<String>` | Возвращает все зарегистрированные имена схем. |
| `typeNames()` | `List<String>` | Возвращает все зарегистрированные имена типов. |
| `close()` | `void` | Вручную освобождает все нативные ресурсы. |

---

:::note
- Нативная библиотека автоматически загружается из упакованных ресурсов во время выполнения.
- Выбрасывается `IllegalStateException` при ошибках разбора/загрузки или при попытке использовать закрытый документ.
:::
