# Справочник API WebAssembly

Библиотека AAM для WebAssembly предоставляет нативную реализацию WASM для высокопроизводительного разбора конфигураций в браузере.

## Экспортируемый класс

| Класс | Описание |
|:---|:---|
| `AAM` | Основной объект для взаимодействия с документом AAM в экосистеме JS. |

---

## Создание

| Конструктор | Возвращает | Описание |
|:---|:---|:---|
| `new AAM(content: string)` | `AAM` | Создает новый документ из строки исходного кода. |

---

## Статический вспомогательный метод

| Статический метод | Возвращает | Описание |
|:---|:---|:---|
| `AAM.lspAssist(content: string)` | `\{ diagnostics: string[]; formatted: string \| null \}` | Инструменты диагностики для реализации языкового сервера. |

---

## Методы экземпляра

| Метод | Возвращает | Описание |
|:---|:---|:---|
| `format(content: string)` | `string` | Форматирует содержимое исходного кода AAM. |
| `formatRange(content: string, startLine: number, endLine: number)` | `string` | Форматирует определенный диапазон строк в коде. |
| `get(key: string)` | `string | undefined` | Прямой поиск по ключу. |
| `keys()` | `string[]` | Возвращает все ключи в документе. |
| `toMap()` | `Record<string, string>` | Преобразует данные библиотеки в JS-объект. |
| `find(query: string)` | `Record<string, string>` | Продвинутый прямой/обратный поиск по ключу или значению. |
| `deepSearch(pattern: string)` | `Record<string, string>` | Рекурсивный поиск ссылок с поддержкой шаблонов. |
| `reverseSearch(value: string)` | `string[]` | Поиск всех ключей, соответствующих заданному значению. |
| `schemaNames()` | `string[]` | Возвращает все зарегистрированные имена схем. |
| `typeNames()` | `string[]` | Возвращает все зарегистрированные имена типов. |
