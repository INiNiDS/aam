# Установка
## Какую библиотеку выбрать?
Вам нужно выбрать библиотеку для вашего языка программирования. Официальные библиотеки:
| Язык | Библиотека | Описание |
|------|------------|----------|
| Rust | [aam-rs](https://crates.io/crates/aam) | AAM-библиотека для Rust |
| Python | aam-py (Скоро) | AAM-библиотека для Python |
На данный момент это всё.
## Установка aam-rs
Чтобы установить `aam-rs`, добавьте следующее в ваш `Cargo.toml`:
```toml
[dependencies]
aam = "1.1.0"
```
Затем вы можете использовать его в вашем коде Rust:
```rust
use aam::aaml::AAML;

fn main() {
    let aaml = AAML::load("config.aam").unwrap();
    println!("{:#?}", aaml.find_obj("key"));
}
```
## Установка aam-py
Чтобы установить `aam-py`, используйте pip:
```bash
pip install aam-py
```
Затем вы можете использовать его в вашем Python-коде:
```python
from aam import AAML

aaml = AAML.load("config.aam")
print(aaml.find_obj("key"))
```
## Документация API
- [Документация aam-rs API](https://docs.rs/aam/latest/aam/)
- [Документация aam-py API](https://aam-py.readthedocs.io/en/latest/)
Функции API в основном одинаковы во всех библиотеках, но могут быть небольшие различия в соглашениях об именах или дополнительных вспомогательных методах. Всегда обращайтесь к документации конкретной библиотеки за подробностями о доступных функциях и их использовании.
