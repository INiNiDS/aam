# AAMのすべての数学型

## ベクトル

| 型 | 文字列キー | 説明 |
|------|-----------|-------------|
| `Vector2` | `"vector2"` | 2Dベクトル (x, y) |
| `Vector3` | `"vector3"` | 3Dベクトル (x, y, z) |
| `Vector4` | `"vector4"` | 4Dベクトル (x, y, z, w) |
