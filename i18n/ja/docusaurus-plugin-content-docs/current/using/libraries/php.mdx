# PHP API リファレンス

PHP 用 AAM ライブラリは、PHP FFI を介して AAM 設定ファイルを操作するための便利なインターフェースを提供します。

## 主要クラス

| クラス | 説明 |
|:---|:---|
| `AamPhp` | AAM ドキュメントを操作するための主要なオブジェクト。 |

---

## 構築

| コンストラクタ | 説明 |
|:---|:---|
| `new AamPhp(?string $libraryPath = null)` | 新しいインスタンスを作成します。ライブラリパスはオプションで、ネイティブ共有ライブラリの場所を明示的に指定するために使用できます。 |

---

## 公開メソッド

| メソッド | 戻り値 | 説明 |
|:---|:---|:---|
| `parseFindObj(string $content, string $query)` | `?string` | 入力コンテンツをパースし、クエリ文字列を使用して `find_obj` 形式の検索を実行します。 |

---

## 実行時の注意

- PHP FFI を使用しており、システムに `aam_rs` ネイティブ共有ライブラリがインストールされている必要があります。
- パスは直接指定するか、環境変数/標準パスから解決されます。
- 提供されたクエリがキーでない場合、逆引き検索へのフォールバックをサポートしています。
