# Go API リファレンス

Go 用の AAM ライブラリは Rust コアライブラリの薄いラッパーであり、効率的な設定パースのためのネイティブ Go インターフェースを提供します。

## 主要な型

| 型 | 説明 |
|:---|:---|
| `AAM` | AAM ドキュメントを操作するために使用される主要な構造体。 |

---

## コンストラクタ

| メソッド | 戻り値 | 説明 |
|:---|:---|:---|
| `New()` | `(*AAM, error)` | 新しい空のドキュメントを作成します。 |
| `Parse(content string)` | `(*AAM, error)` | 文字列からドキュメントを作成します。 |
| `Load(path string)` | `(*AAM, error)` | ファイルパスからドキュメントを作成します。 |

---

## インスタンスメソッド

| メソッド | 戻り値 | 説明 |
|:---|:---|:---|
| `Format(content string)` | `(string, error)` | AAM コンテンツをフォーマットします。 |
| `Get(key string)` | `(string, bool)` | キーによる直接検索。 |
| `Find(query string)` | `map[string]string` | キーまたは値による高度な直接/逆引き検索。 |
| `DeepSearch(pattern string)` | `map[string]string` | パターンをサポートした再帰的な参照検索。 |
| `ReverseSearch(value string)` | `[]string` | 指定された値に一致するすべてのキーを検索します。 |
| `SchemaNames()` | `[]string` | 登録されているすべてのスキーマ名のリストを取得します。 |
| `TypeNames()` | `[]string` | 登録されているすべての型名のリストを取得します。 |
| `LastError()` | `string` | ネイティブハンドルから最後のエラーメッセージを取得します。 |
| `Close()` | `void` | ネイティブハンドルを閉じ、メモリを解放します。 |

---

:::note
- `Close()` は冪等であり、ドキュメントの操作が完了した後に呼び出す必要があります。
- 閉じられたハンドルのメソッドは、特定のメソッドに応じて空の値またはエラーを返します。
:::
