# Referencia de la API C FFI

Fuente de verdad: `include/aam.h`. Esta API proporciona una interfaz C estable para integrar AAM en otros lenguajes o sistemas.

## Ciclo de Vida del Handle

| Función | Descripción |
|:---|:---|
| `AamlHandle *aam_new(void)` | Crea un nuevo handle de AAML. |
| `void aam_free(AamlHandle *handle)` | Libera un handle creado previamente. |

## Analizar y Cargar

| Función | Descripción |
|:---|:---|
| `int aam_parse(AamlHandle *handle, const char *content)` | Analiza el contenido de AAM desde una string. |
| `int aam_load(AamlHandle *handle, const char *path)` | Carga y analiza un fichero AAM desde el disco. |
| `int aam_merge(AamlHandle *handle, const char *content)` | Fusiona nuevo contenido en un handle existente. |
| `int aam_recover_simple(AamlHandle *handle, const char *content)` | Análisis de recuperación seguro para entradas corruptas. |

**Convención de retorno:** `0` en caso de éxito, distinto de cero en caso de error.

## Búsqueda (Lookup)

| Función | Descripción |
|:---|:---|
| `char *aam_find_obj(const AamlHandle *handle, const char *key)` | Búsqueda directa por clave o búsqueda inversa por valor. |
| `char *aam_find_key(const AamlHandle *handle, const char *value)` | Búsqueda inversa explícita. |
| `char *aam_find_deep(const AamlHandle *handle, const char *key)` | Resolución de referencias recursiva. |

:::important
Las strings devueltas son propiedad de Rust y deben liberarse mediante `aam_string_free` para evitar fugas de memoria.
:::

## Memoria y Errores

| Función | Descripción |
|:---|:---|
| `void aam_string_free(char *s)` | Libera una string devuelta por cualquiera de las funciones de búsqueda. |
| `const char *aam_last_error(const AamlHandle *handle)` | Devuelve el último mensaje de error como una string prestada (borrowed). |

:::warning
`aam_last_error` devuelve un puntero interno prestado. **No lo liberes.**
:::
