# What about this category?
This category helps to create your own plugin or library in that programming language, that you need. For example, I need AAM for Rust, and using this category, I made a [library for Rust](https://crates.io/crates/aam-rs). You can also create a plugin for any other programming language, or even for a specific framework. The possibilities are endless!

## Getting started
First of all, you need to check for existing plugins or libraries for your programming language. If there is already a plugin or library for your programming language, you can contribute to it by adding new features, fixing bugs, or improving documentation. If there is no plugin or library for your programming language, you can create one from scratch.
