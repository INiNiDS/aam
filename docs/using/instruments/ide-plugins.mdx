# IDE Plugins

## What use?

IDE plugins provide a better development experience for AAM files. They provide:

- Syntax highlighting
- Autocompletion
- Error checking
- Formatting
- Linting

## Available plugins

| IDE | Plugin | Description |
|-----|--------|-------------|
| VS Code | [aam-vscode](TODO) | AAM plugin for VS Code |
| JetBrains IDEs | [aam-jetbrains](https://plugins.jetbrains.com/plugin/30314-aam-helper/versions/stable/965678) | AAM plugin for JetBrains IDEs |

## Installing

### VS Code

1. Open VS Code
2. Go to the Extensions view (Ctrl+Shift+X or Cmd+Shift+X)
3. Search for "aam"
4. Click "Install"

### JetBrains IDEs

1. Open your JetBrains IDE
2. Go to Settings > Plugins
3. Click "Marketplace"
4. Search for "aam"
5. Click "Install"

## Usage

Once installed, you can use the AAM plugin in your IDE. For example, you can open an `.aam` file and the plugin will automatically provide syntax highlighting and error checking.
