# Installing
## What use?
You need to choose library for your programming language. Authorized libraries:
| Language | Library                                | Description            |
|----------|----------------------------------------|------------------------|
| Rust     | [aam-rs](https://crates.io/crates/aam) | AAM library for Rust   |
| Python   | aam-py (Coming soon)                   | AAM library for Python |
Now it's all.
## Installing aam-rs
To install `aam-rs`, add the following to your `Cargo.toml`:
```toml
[dependencies]
aam = "1.1.0"
```
Then, you can use it in your Rust code:
```rust
use aam::aaml::AAML;

fn main() {
    let aaml = AAML::load("config.aam").unwrap();
    println!("{:#?}", aaml.find_obj("key");
}
```
## Installing aam-py
To install `aam-py`, you can use pip:
```bash
pip install aam-py
```
Then, you can use it in your Python code:
```python
from aam import AAML

aaml = AAML.load("config.aam")
print(aaml.find_obj("key"))
```
