# All Math Types in AAM

## Vectors

| Type | String Key | Description |
|------|-----------|-------------|
| `Vector2` | `"vector2"` | 2D vector (x, y) |
| `Vector3` | `"vector3"` | 3D vector (x, y, z) |
| `Vector4` | `"vector4"` | 4D vector (x, y, z, w) |
