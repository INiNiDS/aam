Referência de API aam-rs

aam-rs é a biblioteca Rust oficial para analisar, validar e construir ficheiros de configuração AAM. A biblioteca Python aam-py expõe a mesma API — ambas as linguagens são mostradas em cada exemplo.

Nota: Adicione aam ao seu Cargo.toml (Rust) ou execute pip install aam-py (Python) antes de usar esta API. Consulte a página Instalação para mais detalhes.

AAML​

O ponto de entrada principal para todas as operações AAM. AAML contém o mapa de chave-valor analisado, definições de tipo registadas e definições de schema. Todas as diretivas (@derive, @schema, @import, @type) são processadas automaticamente em tempo de análise.

Construtores​

AAML::new​

Cria uma nova instância AAML vazia com todos os manipuladores de diretivas padrão registados.

use aam_rs::aaml::AAML;
let cfg = AAML::new();
from aam import AAML
cfg = AAML()

AAML::parse​

Analisa uma string AAM e retorna uma nova instância AAML totalmente populada.

ParâmetroTipoDescrição
content&strTexto de origem AAM a analisar.

Retorna: Result<AAML, AamlError>

let cfg = AAML::parse("host = localhost\nport = 8080").unwrap();
assert_eq!(cfg.find_obj("host").unwrap().as_str(), "localhost");

AAML::load​

Lê um ficheiro AAM do disco e retorna uma nova instância AAML.

ParâmetroTipoDescrição
file_pathimpl AsRef<Path>Caminho para o ficheiro .aam.

Retorna: Result<AAML, AamlError>

let cfg = AAML::load("config.aam").unwrap();
println!("{}", cfg.find_obj("host").unwrap());

Fusão​

merge_content​

Analisa texto de origem AAM e funde os pares chave-valor resultantes na instância atual.

let mut cfg = AAML::new();
cfg.merge_content("host = localhost").unwrap();
cfg.merge_content("port = 8080").unwrap();

merge_file​

Lê um ficheiro do disco e funde o seu conteúdo na instância atual.

let mut cfg = AAML::new();
cfg.merge_file("base.aam").unwrap();
cfg.merge_file("overrides.aam").unwrap();

Combinar Instâncias​

Duas instâncias AAML podem ser combinadas usando os operadores + e +=. O lado direito vence em conflitos de chave.

let base = AAML::parse("host = localhost\nport = 80").unwrap();
let overrides = AAML::parse("port = 443").unwrap();
let combined = base + overrides;
assert_eq!(combined.find_obj("port").unwrap().as_str(), "443");

Pesquisa​

find_obj​

O método de pesquisa principal. Pesquisa key no mapa. Se não for encontrada nenhuma chave correspondente, realiza uma pesquisa inversa automática.

Retorna: Option<FoundValue>

let cfg = AAML::parse("a = b").unwrap();
assert_eq!(cfg.find_obj("a").unwrap().as_str(), "b");
assert_eq!(cfg.find_obj("b").unwrap().as_str(), "a"); // pesquisa inversa

find_key​

Pesquisa inversa explícita. Encontra a chave cujo valor mapeado é igual a value.

Retorna: Option<FoundValue>

find_deep​

Segue uma cadeia de pesquisas key → value → key até um valor terminal ser atingido ou um ciclo ser detetado.

Retorna: Option<FoundValue>

let cfg = AAML::parse("a = b\nb = c").unwrap();
assert_eq!(cfg.find_deep("a").unwrap().as_str(), "c");

Validação de Schema​

get_schema​

Retorna o SchemaDef registado com name, ou None.

apply_schema​

Valida um mapa data contra um schema nomeado.

validate_schemas_completeness​

Verifica cada campo obrigatório de cada schema registado contra o mapa de chave-valor atual.

Retorna: Result<(), AamlError>

FoundValue​

O tipo retornado por todos os métodos de pesquisa AAML. Encapsula o valor de string bruto e fornece helpers para transformações comuns.

Métodos​

MétodoRetornaDescrição
as_str()&strRetorna o valor interno como um slice de string.
as_list()Option<Vec<String>>Analisa o valor como uma lista AAM [...].
as_object()Option<HashMap<...>>Analisa o valor como um objeto inline {...}.
is_list()boolVerdadeiro se o valor é um literal de lista.
is_object()boolVerdadeiro se o valor é um objeto inline.
remove(target)&mut SelfRemove todas as ocorrências de target.

AamlError​

Todos os erros produzidos pela pipeline de análise e validação implementam std::error::Error e Display.

Variantes​

VarianteDescrição
IoError(io::Error)Ocorreu um erro de I/O ao ler um ficheiro.
ParseError { line, content, details }Uma linha de origem não pôde ser analisada.
NotFound(String)Uma chave ou nome de tipo não foi encontrado.
InvalidValue(String)Um valor falha uma restrição de tipo básica.
SchemaValidationError { schema, field, type_name, details }Uma restrição de schema foi violada.
match AAML::load("config.aam") {
    Ok(cfg) => println!("Carregado: {:?}", cfg),
    Err(AamlError::IoError(e)) => eprintln!("Não foi possível ler o ficheiro: {e}"),
    Err(AamlError::ParseError { line, content, details }) => {
        eprintln!("Erro de análise na linha {line}: '{content}' — {details}");
    }
    Err(e) => eprintln!("Erro: {e}"),
}
